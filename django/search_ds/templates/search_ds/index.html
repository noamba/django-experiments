In this small (and still developing) project I wanted to:

  • Use Python, Django, MySQL, Javascript, jQuery, HTML, CSS, Bootstrap etc. with Git to explore these technologies and experiment with them
  • Have this website render nicely on all devices including mobiles and tablets
  • Create a web service json API
  • Deploy the outcome to the web

I started with a dataset that includes about 70,000 clothing items in csv format of id, description, brand (view part of it here or download it here).

The following steps were to:

  • Create a Django site and app (mysite and search_ds, see here)
  • Create a MySQL database
  • Transform the datatset to normalized Django models (see models)
  • Create objects and save in the MySQL database (see the script that does that)
  • Create a url and a view to aggregate, order and filter the brands with the most items, and,
  • Present these brands in a dynamic bar chart using Javascript, jQuery and Google Charts (see here), HTML, Bootstrap and CSS (see base and graph templates). And, finally
  • Deploy to Pythonanywhere

See the RESULT and check it out: hover over the bars and click on the buttons.

For the web service json API, I wanted to provide the top companies and their corresponding total items. See the view and the RESULT. Remember to use the back arrow in your browser to get back to this page.
