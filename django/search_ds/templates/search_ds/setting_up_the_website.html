Setting up the website

Very breifly, the steps to set up this website were:

  1. Install Virtualenv and Git.
  2. Set up a Python 3 and Django 1.9 environment.
  3. Create a Django project and app and push to Github (see mysite and search_ds here).
  4. Install a MySQL database.
  5. Use Bootstrap and its Grid system to have the web pages responsive (i.e. render differently on different devices), see example here.
  6. And finally, deploy from Github to Pythonanywhere.
